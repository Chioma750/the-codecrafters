## String Transformer
This project is designed in a way that the string transformer takes in corrupted inputs and produce clean correctly and formatted output.

## Running the Application (Usage)
To run the project you need to type "go run main.go" on your terminal.
Choose a word or a sentence of your choice you want to transform, choose any of the input transformations listed to you, then hit the enter botton.
You will get a correctly formatted output.

## Note That
If you select any operator that is not from the listed given to you, you will not get a correctly formatted output, rather it will print Unrecognised.
