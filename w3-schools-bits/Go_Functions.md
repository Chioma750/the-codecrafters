## Understanding Go Functions
A Go function is a reusable, self-contained module of code designed to perform a specific task. Rather than running instantly, functions are stored and executed only when explicitly invoked. 

### Declaring a Function
To define a function, use the func keyword, provide a name, include parentheses (), and place the executable code within curly braces {}.

### Executing (Calling) a Function
Functions are executed by referencing their name followed by parentheses (). They can be called multiple times throughout a program. 

### Function Naming Rules

* Case-Sensitivity: Function names are case-sensitive.
* Start Character: Must begin with a letter.
* Allowed Characters: Alpha-numeric characters and underscores are allowed (A-z, 0-9, and _).
* Spaces: No spaces are permitted.
* Naming Convention: For multi-word names, use MixedCaps or mixedCaps rather than underscores.
* Best Practice: Choose descriptive names that reflect the function's purpose.
