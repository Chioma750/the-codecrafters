## Constant
If a variable has a fixed value that cannot be changed, you can use the **const** keyword.

The **const** keyword declares the variable as "constant" which means that it is unchangeable and reads only. 

## Constant Rules
1. Constant names follow thesame naming rules as variable.
2. Constant names are usually writting in capital letters (for easy identification    and to differenciate it from variable).
3. Constant can be declared on both inside and outside of a function

## Constant Types
    There are two types of constant which are :
    1. The Typed constant
    2. The Untyped constant

**Explanation**

    1. Typed constant are declared with a defined type  E.g
    const A int = 1
    2. Untyped constant are declared without a type  E.g
    const A = 1

**NB**: In this case the compiler decides the type of the constant, based on the value.
