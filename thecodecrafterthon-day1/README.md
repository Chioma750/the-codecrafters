## Cli-Calculator
This project is a calculator project that runs in a terminal, accepts user input and responds with a result.

## Running the Application (Usage)
To run the project you need to type "go run main.go" on your terminal.
Enter input1 (any digit of your choice), select any operator from the listed given to you, then enter input2 and hit the enter botton.
You will get the total sum of what you inputed.   

## Note That:
If you select any operator that is not from the listed given to you, you will not get the total sum of what you inputed, rather it will print out "Invalid operation".
