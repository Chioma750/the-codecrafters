## The Base Converter
This project is all about number systems and string convertion. It is a CLI tool that converts numbers between bases.

## Running the Application (Usage)
To run this project you need to type "go run main.go" on your terminal.
Enter any word of your choice that you want to convert, choose input bases that you want your word to convert to from the listed given to you, then hit the enter botton.
You will get the correct output.

## Note That
If you select a wrong word to be converted, or an input base that is not from the listed given to you, you will not get the correct output, rather it will print "Invalid Decimal".
